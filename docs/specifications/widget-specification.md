# Widget specification

The prompt-created widget is version `1.0` and supports governed DQI usage, decision, policy, assessment, risk, grounding, and integration metrics; integration/model/environment/policy/decision/severity dimensions; allowlisted named filters; complete-week ranges; five chart types; four palettes; light/dark themes; query inspection; and provenance. Widget evolution will add compound boolean filters, comparisons, thresholds, drill-down, ownership, immutable published versions, and reviewed regulatory mappings.

**Related:** WIDGET-001.
