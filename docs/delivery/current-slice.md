# Current slice: natural-language EU AI Act audit widget

## User outcome

A user describes an EU AI Act-oriented KPI, dimension, time window, chart, palette, and theme. DQI displays the resulting semantic plan and Elasticsearch DSL, calculates the result from a realistic synthetic audit event store, and renders a compliance widget with provenance.
